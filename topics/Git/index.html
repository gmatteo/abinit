    Introduction¶

    ABINIT is developed under the control of git and the development is based on the Gitflow branching model. There is an official github mirror where we welcome bug fixes and improvements. Note, however, that most of the active developments are hosted on our internal Gitlab server. Before embarking on making significant changes, please contact the Abinit group.

    See the many different projects linked to ABINIT on Github actually gathered in the Github ABINIT organization.
